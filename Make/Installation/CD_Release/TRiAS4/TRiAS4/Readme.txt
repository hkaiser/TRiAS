TRiAS GmbH Potsdam, TRiAS V4.0, Potsdam, den 8. Februar 2001
Letzte Informationen

In dieser Datei sind folgende Punkte enthalten:
	1. Systemvorraussetzungen
	2. Installation
	3. Handbuch
	4. Auftretende Fehler


1. Systemvorraussetzungen für TRiAS4
- WindowsNT V4.0 (SP3) oder Windows 95/98 oder neuer
- Internet-Explorer V4.0 (SP1) oder neuer. Sie können den Internet-Explorer V5.5 
direkt von der vorliegenden CD installieren. Bitte starten Sie dazu im Verzeichnis 
\IE55\i386 das Programm ie5setup.exe.


2. Installation von TRiAS
- Zur Installation starten Sie bitte die Datei TRiAS4.exe im 
Verzeichnis \TRiAS4 der CD. 
- Unter WindowsNT und WIndows 2000 muß TRiAS V4.0 z.Zt. von einem Nutzer installiert 
werden, welcher Administratorrechte besitzt. Die Installation wird anderenfalls 
abgebrochen.
	

3. Handbuch
- Das komplette Handbuch (derzeit das Handbuch der TRiAS V3.0) ist im Verzeichnis 
\Handbuch dieser CD enthalten. Das aktuelle Handbuch für die Version 4.0 befindet sich 
derzeit in Überarbeitung und wird demnächst fertiggestellt. Bitte wenden Sie sich
an uns (TRiAS GmbH Potsdam, Dennis Gabor Str. 2, 14469 Potsdam, 
email TRiAS@intertrias.com), wenn Sie interesse an der überarbeiteten Version des 
Handbuches haben.
- Eine Beschreibung des Lizensierungsverfahrens von TRiAS4 finden Sie im Verzeichnis
\Handbuch dieser CD im PDF Format.


Das Handbuch liegt im PDF-Format vor. Sollten Sie keine Möglichkeit haben, dieses Format
anzuzeigen, so sollten Sie das Program 'Acrobat Reader' installieren, dessen
Installationsprogramm sich im Verzeichnis \AcroRead befindet.


4. Auftretende Fehler
Wir möchten Sie bitten bei auftretenden Fehlern im Programm oder Handbuch bzw. 
Unstimmigkeiten zwischen beiden an uns ein Fehlerprotokoll zu senden.
