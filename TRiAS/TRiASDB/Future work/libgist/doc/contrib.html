Contributors

  • The Generalized Search Tree was originally described in a paper by Hellerstein, Naughton and Pfeffer which appeared in the Proceedings of the VLDB '95 conference.
    • An initial implementation based on the concepts in that paper was performed by Joe Hellerstein and Shouyi Hsiao.
  • The code and documentation in libgist v.2.0 and v.1.0 was developed by Marcel Kornacker, Paul M. Aoki and Mehul Shah.
  • The analysis framework implemented in amdb is described in the paper "An Analysis Framework for Access Methods," which can be downloaded from this page.
  • The code and documentation in amdb v.1.0 was developed by Marcel Kornacker and Mehul Shah.
  • Additional contributions to the documentation have been made by Megan Thomas.
Comments, questions and suggestions may be directed to gist@postgres.berkeley.edu
